Venha conheçer nossa cafeteria

Localizada no coração de São Paulo, a Café Aroma & Sabor é um refúgio para os amantes de café. Com uma decoração aconchegante e uma seleção cuidadosa de grãos especiais, esta cafeteria oferece uma experiência única. Ao entrar, você é recebido pelo aroma inebriante dos grãos recém-moídos. O balcão exibe uma variedade de métodos de preparo, desde a clássica prensa francesa até o moderno aeropress. Os baristas, apaixonados pelo seu ofício, estão sempre prontos para criar a bebida perfeita para você. O cardápio inclui clássicos como o cappuccino e o espresso, mas também surpreende com combinações inusitadas, como o “Café Caramelo” (um espresso duplo com xarope de caramelo e chantilly). Para acompanhar, experimente os bolos caseiros ou os pães de queijo fresquinhos. Seja para uma pausa rápida ou para uma tarde relaxante, a Café Aroma & Sabor é o lugar ideal para apreciar o melhor que o café tem a oferecer.
